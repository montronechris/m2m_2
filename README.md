# Tavola Rapida 🍝

Sistema moderno di gestione ordini e tavoli per ristoranti.

Sviluppato con **Next.js 16 (App Router)**, **Supabase** (Backend + Auth + Database), **TypeScript** e **Tailwind CSS**.

## ✨ Funzionalità principali

- Gestione tavoli e stato (libero/occupato)
- Creazione e gestione ordini in tempo reale
- Sistema di autenticazione (admin / camerieri)
- Dashboard intuitiva
- Aggiornamenti live con Supabase Realtime

## 🚀 Come installare

```bash
# Clona il repository
git clone https://github.com/montronechris/m2m-da-vedere.git
cd m2m-da-vedere

# Installa le dipendenze
npm install

# Configura le variabili d'ambiente
cp .env.example .env.local
